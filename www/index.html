Ruby/CVS - CVS library for Ruby

Ruby/CVS provides high level interface to a CVS repository.

Requirements

  • ruby-1.6.4 (older version may work.)
  • cvs-1.11.1p1 (older version may work.)
  • flex.rb-0.10 (optional. older version doesn't work.)
